# ⚙️ Anime Repo — Workflow Rules

## 📓 KNOWLEDGE BASE SAVE PATTERN

When saving a knowledge base file, use this EXACT pattern:

```
Save as: C:\Anime\AlphaBabes\knowledge_base\<NUMBER>_<short-title>.md
Then run: git add . && git commit -m "KB: <short-title>" && git push
```

- Files are numbered with gapped tens (000, 010, 020...) in reading order.
- Short title is lowercase, hyphenated.
- Commit + push after EVERY kb file save. NEVER skip.
